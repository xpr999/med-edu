# 🏥 藥物衛教系統 — 部署說明

## 檔案結構
```
med-edu/
├── api/
│   └── analyse.js        ← 後端（呼叫 Gemini API）
├── public/
│   └── index.html        ← 前端網頁
├── vercel.json           ← Vercel 設定
└── package.json
```

---

## 📋 部署步驟（約 15 分鐘）

### Step 1：申請 Google Gemini API Key（免費）

1. 開啟 https://aistudio.google.com
2. 登入 Google 帳號
3. 左側點「Get API Key」→「Create API Key」
4. 複製這串 Key（AIzaSy...開頭），先存到記事本

> 免費額度：每天 1,500 次呼叫，每分鐘 15 次，對診所使用量完全夠用

---

### Step 2：把專案上傳到 GitHub

1. 開啟 https://github.com → 登入
2. 右上角「+」→「New repository」
3. Repository name: `med-edu`
4. 選 **Private**（私人）→ 點「Create repository」
5. 按照 GitHub 頁面上的指示，把這個資料夾的檔案上傳上去

   **最簡單的方法（網頁上傳）：**
   - 點「uploading an existing file」
   - 把整個 med-edu 資料夾內的所有檔案拖進去
   - 注意：api/ 和 public/ 資料夾要分別上傳，保持資料夾結構

---

### Step 3：部署到 Vercel（免費）

1. 開啟 https://vercel.com → 用 GitHub 帳號登入
2. 點「Add New Project」→ 選剛才建立的 `med-edu` repository
3. 點「Deploy」（不用改任何設定）
4. 等待部署完成（約 1 分鐘）
5. 你會得到一個網址，例如 `https://med-edu-xxx.vercel.app`

---

### Step 4：設定 Gemini API Key

部署完成後，**一定要做這步**，否則 AI 功能不會運作：

1. 在 Vercel 專案頁面，點上方「Settings」
2. 左側點「Environment Variables」
3. 點「Add New」
   - Name: `GEMINI_API_KEY`
   - Value: 貼上你的 API Key（AIzaSy...）
   - 點「Save」
4. 回到「Deployments」→ 點最新一筆右邊的「⋯」→「Redeploy」

---

### Step 5：開始使用 ✅

1. 打開你的網址：`https://med-edu-xxx.vercel.app`
2. 截圖診所系統（處方的畫面）
3. 上傳截圖
4. 點「辨識並生成衛教單」
5. 列印給病患

## 🔧 日常維護

### 更新藥物資料庫
編輯 `api/analyse.js` 中的 prompt，加入你診所的特殊藥物代碼

### 費用估算
- Vercel 托管：**免費**
- Gemini API：**免費**（每天 1,500 次，超過才收費，但很便宜）

### 安全性
- API Key 存在 Vercel 環境變數，不會外洩
- 建議在 Vercel 設定中限制網域（只允許你診所的 IP 存取）

## ❓ 常見問題

**Q: 辨識結果不準確？**
A: 在「補充說明」欄位加入更多描述，例如「這是精神科處方，使用台灣健保碼」

**Q: 截圖辨識不到藥名？**
A: 確認截圖清晰，並包含藥物名稱欄位。可試著框選只有藥物表格的部分

**Q: 想加入診所 logo 或名稱到衛教單？**
A: 告訴 Claude，可以幫你在 index.html 加入診所資訊
